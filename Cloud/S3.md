# S3(Simple Storage Service)

**객체기반** 클라우드 스토리지 서비스(데이터저장및관리)

S3 = 버킷: 파일을 담는 그릇

리전범위의 객체 저장소

특징
- 높은 내구성: Region에 복사본이 많아 오류 날 일이 거의 없음
- 많은 기능
- 신뢰성
- 확장성
- 관리 유용

버킷정책에는 버킷 단위로 정해지는 정책으로   
Principal이 들어간다.

[정책생성기](https://awspolicygen.s3.amazonaws.com/policygen.html)에서 생성가능

버전관리 가능   
활성화 하여 사용, 최신버전을 지속적으로 S3에 저장 중   


## Glacier
장기 백업 가능   
비용은 저렴함   
Glacier(빙하)라는 이름 처럼 녹이는데 시간이 걸림 -> 복원하는 시간이 오래걸림
