# VPC 

AWS 에서 생성할 수 있는 프라이빗 가상 네트워크   
VPC 사설 네트워크 안에서 EC2를 배포

### AWS Direct Connect

전용회선으로 퍼블릭 통신망을 쓰지않고 온프레미스와  AWS간에 통신을 할 수 있음

장애를 대비하기위해 예산이 된다면 이중으로 설치 하기도 함 or VPN구성

### VPC 피어링

IGW를 거치지 않고 VPC와 VPC끼리 Private하게 연결 가능

VPC가 늘어남에 따라 전이적 피어링의 한계를 극복하기 위해 TGW서비스가 나옴

CIDR주소가 같으면 연결이 안되므로 VPC 초기 설정 시 피어링을 고려하여 주소범위를 고려해야함

### VPC엔드포인트
인스턴스와 AWS서비스 간에 NAT를 거치지 않고 Private하게 통신할 수 있는 서비스

사용가능 범위가 점점 확장되고 있음
