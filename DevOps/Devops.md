# DevOps

애플리케이션과 서비스를 빠른 속도로 제공할 수 있도록 조직의 역량을 향상시키는 문화 철학, 방식 및 도구의 조합 - 출저: [AWS-What-Is-DevOps](https://aws.amazon.com/ko/devops/what-is-devops/)

조직의 소프트웨어 제품과 서비스를 빠른 시간안에 개발 및 배포

DevOps툴체인 방법론 

CI/CD

지속적 통합과 지속적 배포

# DevSecOps

설계, 개발, 테스트, 생산, 운영까지 애플리케이션 라이프 사이클 전반에 보안을 통합

보안분야 중 개발자와 가장 밀접한 분야

SAST - 정적 애플리케이션 보안 테스트
 
DAST - 동적 애플리케이션 보안 테스트

CSPM - 클라우드 보안 형상 관리
