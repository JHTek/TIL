# 사이버 킬 체인
```c
 // P.O.V ) 사이버 킬체인에 대한 설명과 모의해킹과의 관계에 대한 본인의 주관적 의견
```

사이버 킬체인은 보안팀에서 사이버 공격을 식별하고 중지할 수 있도록 지원하는 사이버 보안 모델로 총 7단계를 따라 수행됩니다.

1단계는 정찰 단계로, 직원들의 개인 SNS 또는 Shodan을 통해 기업의 취약점에 대한 정보를 조사하거나 Nmap, Burp Suite 등의 정찰 도구로 정보를 수집합니다.

2단계는 무기화 단계로, 침투시킬 멀웨어를 제작하는 단계입니다.

3단계는 전달 단계로, 이전 단계에서 제작한 멀웨어를 침투시키는 단계입니다. 주로 피싱, 악성 USB를 통해 멀웨어가 전달 됩니다.

4단계는 취약점 공격 단계로, 침투한 네트워크에서 악성코드의 실행권한을 얻는 단계입니다.

5단계는 설치 단계로, 공격자가 다시 돌아올 수 있도록 백도어를 설치하는 등 악성도구를 설치하는 단계입니다.

6단계는 C2 단계로, C2서버를 외부에 구축하여 내부 악성코드에게 명령을 수행하게 합니다. 이 단계에서 원격제어나 아웃바운드 신호를 속이며 통신합니다.

7단계는 목표실행 단계로, 측면이동 공격을 하며 데이터를 유출하고 랜섬웨어를 실행하는 등 공격의 실질적 실행 단계입니다.

사이버 킬체인은 단편적인 취약점에 대한 분석을 넘어 각단계에 따르는 시나리오를 따라가기 때문에 APT 공격에 대한 사건흐름을 느낄 수 있습니다.

모의해킹에서 이 모델을 가이드 라인으로 삼으면, 공격을 수행한다면 각 단계에서의 취약점을 자세하게 파악할 수 있고, 실제 공격에 대해 어느지점에서 어떤 조치를 취해 피해를 대비할지 알 수 있습니다.

결과적으로 사이버 킬체인은 살제 위혁에 있어서 대응 체계를 구축하고 방어 우선순위를 설정하는 데 있어 모의해킹의 중요한 지침으로 사용됩니다.
