# TCP/IP Protocol Stack
OSI 7계층보다 실사용이 많이되는 프로토콜 계층이다.

7~5계층을 합해 _Application Layer_ 라 부르며 이 구간에는 
- FTP
- SSH
- Telnet
- HTTP
- HTTPS
- DNS
- TFTP
- SNMP
  
가 포함된다.

4계층은 _Transport Layer_ 라 부르며 TCP와 UDP가 포함된다.

3계층은 _Internet Layer_ 로 IP구간이다.

2계층은 _Network Access Layer_ 로 LAN과 WAN이 포함된다.

TCP헤더의 목적지의 port번호가 _SAP(Service Access Point)_ 이다.
