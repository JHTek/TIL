> # Noting rule

- 너무 **자세하게는 설명 하지 않는다.** 내가 보고 바로 이해할 정도를 지향하기 때문이다. 내가 배우고 이해한 스토리 라인을 따라 기록한다.

- 이름은 TIL 이지만 하루 한 번 정리는 아니다. 하루에도 여러번 저장할 수 있으며 쉴 때는 쉬고 한다. 숙제처럼 무리하게 내용만 붙여넣는 걸 방지하기 위함이다.

- **Issues** 혹은 **PR**을 통한 질문과 교정은 언제나 환영입니다.
